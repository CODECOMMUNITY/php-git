Git\Index::add¶

add specified file to your index. you have to call Git::Index::write after this method.

Description¶

public Git\Index::add (string $path,int $staging = 0)

Parameters¶

path
file path.

Return Values¶

Examples¶

<?php
$repo = new Git\Repository("/tmp/sample/.git");
$index = $repo->getIndex();
$index->add("newcontents.txt");
$index->write();
$index->refresh();
